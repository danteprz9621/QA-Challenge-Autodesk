# Email Verifier

This repository contains code for validating email addresses with a Python-based EmailVerifier class and associated tests.

## Contents

    EmailVerifier.py: Contains the EmailVerifier class for validating email addresses.
    EmailVerifierTest.py: Contains unit tests to ensure the correctness of the verifier.
    README.md: This file.

## Installation

    Clone this repository using:
	 git clone https://github.com/danteprz9621/QA-Challenge-Autodesk.git
    Install the required dependency: 
	 pip install assertpy

## Running Tests

    Navigate to the repository directory in your terminal.
    Run the tests using pytest: 
	 pytest -v EmailVerifierTest.py
